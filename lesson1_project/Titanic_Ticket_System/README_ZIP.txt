## 🎯 What This Program Does

1. ✅ Loads historical Titanic passenger data (1,309 records)
2. ✅ Collects your passenger information (name, age, fare, gender)
3. ✅ Validates all inputs with error checking
4. ✅ Assigns you to a cabin class (1st, 2nd, or 3rd)
5. ✅ Generates a unique 6-digit ticket number
6. ✅ Calculates your survival probability
7. ✅ Saves your ticket to a text file
